# my-first-contribution

Learn how to create a Pull Request (PR) and make your first Open Source contribution step by step!

## How does it work?

In most Open Source projects, the way to contribute is described in a file called `CONTRIBUTING.md`.

In this project, your goal is to add your GitHub username to the `CONTRIBUTORS.md` file by creating a Pull Request.

Please read the [`CONTRIBUTING.md`](CONTRIBUTING.md) file carefully, and then make your first contribution!

You won't be judged here. We all start somewhere!

> [!NOTE]
> There are two subtleties in this project that you need to find to get your Pull Request accepted.
If you find them, you'll be able to successfully add your GitHub username to the `CONTRIBUTORS.md` file.
> If not, your PR will be closed, but you're welcome to try again ♡
